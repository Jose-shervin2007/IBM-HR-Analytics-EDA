# IBM HR Analytics Employee Attrition - Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the IBM HR Analytics Employee Attrition & Performance dataset.

The objective is to identify patterns, trends, and factors that influence employee attrition using statistical summaries and data visualizations.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

1. Import Libraries
2. Load Dataset
3. Dataset Overview
4. Data Cleaning
5. Univariate Analysis
6. Bivariate Analysis
7. Correlation Analysis
8. Outlier Detection
9. Key Findings
10. Business Recommendations
11. Conclusion

## Dataset

IBM HR Analytics Employee Attrition & Performance Dataset
